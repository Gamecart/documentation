Gamecart is a store platform for game server owners. Use it to sell digital products, accept payments, deliver purchases through your server integration, and manage storefront operations from one dashboard.

  This documentation starts with setup guides. API and webhook references are intentionally lightweight until they are generated or verified from the current backend contract.
</Tip>
